# BNPL E-Commerce Platform

A modern Buy Now Pay Later e-commerce platform built with Next.js, React, TypeScript, and Stripe integration with Klarna BNPL support.

## Features

- 🛍️ Product catalog with filtering and search
- 🛒 Shopping cart management
- 💳 Stripe payment integration
- 📅 Klarna Buy Now Pay Later (BNPL) option
- 👤 User authentication
- 📦 Order management
- 📱 Fully responsive design
- ⚡ Server-side rendering for optimal performance
- 🔒 Secure payment processing

## Tech Stack

- **Frontend**: Next.js 14, React 18, TypeScript, Tailwind CSS
- **Payment**: Stripe, Klarna
- **State Management**: Zustand
- **HTTP Client**: Axios

## Getting Started

### Prerequisites

- Node.js 18+ and npm/yarn
- Stripe account (get keys from [Stripe Dashboard](https://dashboard.stripe.com))
- Klarna account (get credentials from [Klarna Merchant Portal](https://merchants.klarna.com))
